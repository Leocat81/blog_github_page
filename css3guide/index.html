# box-shadow

box-shadow: offset-x offset-y blur spread color inset;

box-shadow: X轴偏移量 Y轴偏移量 [阴影模糊半径] [阴影扩展] [阴影颜色] [投影方式];

单词解释：blur:模糊 spread:伸展 inset:内凹

阴影和布局

  • 阴影不影响布局， 但是可能会覆盖其他 box 或者其他 box 的阴影。
  • 阴影不触发滚动条，也不增加滚动区域的大小。
  • 所以布局时可忽略阴影。
